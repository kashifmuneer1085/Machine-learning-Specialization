# Machine Learning Specialization

This repository contains my notes, practice labs, assignments, and projects from the Machine Learning Specialization by Andrew Ng.

## About the Course

The Machine Learning Specialization is designed to provide a strong foundation in machine learning concepts and practical implementation using Python.

Topics covered include:

- Supervised Learning
- Unsupervised Learning
- Neural Networks
- Regression and Classification
- Decision Trees
- Clustering
- Reinforcement Learning Basics
- Model Evaluation
- Machine Learning Best Practices
